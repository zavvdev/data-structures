# Linked List

A linked list is a linear data structure where elements, called nodes, are stored non-contiguously in memory, unlike arrays. Each node contains data and a pointer (or link) to the next (or also previous) node in the sequence. This structure allows for efficient insertion and deletion of nodes, as only pointers need updating, rather than shifting elements
